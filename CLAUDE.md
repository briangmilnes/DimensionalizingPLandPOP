# DimensionalizingPLandPOP

project-tz: America/Los_Angeles

This project dimensionalizes programming languages, program logics, proof logics,
proof assistants, and proofs of programs — cataloging the axes along which each
varies and placing concrete systems on them. The docs live in `docs/`.

This project adopts the ComputAItionalThinking ruleset so agents think and speak in
computer science terms and avoid analogies and metaphors. Any other project may adopt
the same ruleset by adding the identical import line below to its own `CLAUDE.md`.

@~/projects/ComputAItionalThinking/ComputAItionalThinkingRules.md

naming) for orchestrating multi-agent work, imported by reference below.
